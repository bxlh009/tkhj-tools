IELTS General Training Reading allows 60 minutes for three sections and includes answer transfer within that hour. The sections increase in difficulty, so an equal twenty-minute split is a useful starting point, not an unbreakable rule.

## Build a baseline before choosing times

Take one official sample under the full time limit. Record the finish time and accuracy for each section. Your plan should respond to that evidence.

Use a simple table on paper:

| Section | Minutes used | Correct | Unanswered | Main delay |
| --- | ---: | ---: | ---: | --- |
| 1 |  |  |  |  |
| 2 |  |  |  |  |
| 3 |  |  |  |  |

If Section 1 already takes 25 minutes, promising yourself more time for Section 3 will not solve the problem. You need a stop rule.

## Start with a flexible allocation

A practical first experiment is:

1. Section 1: 15 minutes;
2. Section 2: 18 minutes;
3. Section 3: 22 minutes;
4. final transfer and spelling check: 5 minutes.

This is not an official recommended split. It is a testable plan that reserves more time for the longer, more complex final text and protects answer transfer.

Adjust in two-minute increments after reviewing two full practices. Large changes make it hard to learn what helped.

## Use a two-pass system

On the first pass, answer items whose evidence you can locate and verify. Mark a stalled item and move on when you have reread the same lines without gaining information.

On the second pass, return with a specific question: am I missing a paraphrase, a limiter, or the relevant location? Repeating the same search is not a new attempt.

For paper practice, circle the question number. For computer practice, use the review feature available in the official interface. Your method should match the delivery mode you will take.

## Match reading behavior to the task

Do not apply “questions first” as a universal rule. It works well for locating names, dates, conditions, and short factual details. For matching headings or identifying a writer’s purpose, a quick paragraph map may be more efficient.

Choose the reading action from the question:

- detail or short answer: scan for a unique anchor, then read locally;
- True/False/Not Given: locate the claim and compare meaning precisely;
- matching headings: identify each paragraph’s main job;
- completion task: predict the required grammar before searching.

The fastest method is the one that collects the right evidence, not the one that moves your eyes fastest.

## Create recovery rules

A time plan fails when it has no response to difficulty. Write three rules before practice:

1. If one item passes my time limit, mark it and move.
2. If a whole text feels difficult, answer the easiest visible item first.
3. If five minutes remain, complete answer transfer before returning to uncertainty.

These rules reduce the chance that one problem consumes the time intended for an entire section.

## Protect accuracy at the end

The official format warns that spelling and grammar can affect answers and that no extra Reading transfer time is provided. Your final check should therefore focus on mechanical losses:

- every question has an answer;
- word limits are followed;
- singular and plural forms match the text;
- names and numbers are copied accurately;
- answers are in the correct boxes.

Do not use the final minute to rethink a large group of answers without evidence.

## Review the plan, not only the score

After practice, ask which rule you followed and where the plan broke. If accuracy was high but questions were unanswered, work on stop rules. If everything was completed but accuracy collapsed, slow the evidence check. If one task type consumed time, isolate it in short drills before the next full test.

A useful time plan is flexible across sections and strict about recovery. It allocates attention where the evidence says you need it.
