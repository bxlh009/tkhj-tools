一次练习分数只能告诉你曾经发生过什么。错题本应该改变你接下来的行动。如果它只记录题号和正确答案，它就变成了档案库，而非学习系统。

## 记录决策，而非尴尬

为每一个错误答案和你曾犹豫不决的正确答案各设一行。

| 字段 | 填写内容 |
| --- | --- |
| 任务 | 具体的题目或写作技能 |
| 我的决策 | 我选择的内容或写下的内容 |
| 证据 | 控制答案的文本、规则或描述符 |
| 错误类型 | 导致决策失败的原因 |
| 修复规则 | 下一次尝试时的一个具体行动 |
| 复习日期 | 你何时将再次测试该规则 |

保持语言中立。“粗心”无法解释一个决策。“忽略了单词 *only* 并接受了更宽泛的主张”则可以。

## 使用小型错误分类法

从六个标签开始：

1. 误解了任务；
2. 无法定位证据；
3. 误读了限制词或关系；
4. 使用了课外知识；
5. 知道规则但执行错误；
6. 时间耗尽。

只有当现有标签确实无法描述该错误时，才添加新标签。类别过多会掩盖模式。

## 编写可观察的修复规则

一条强有力的修复规则以行动和触发条件开头。

弱：“提高词汇量。”  
更强：“当目标词汇不熟悉时，在查看选项之前，标记其语法角色和最近的对比词。”

弱：“更好地管理时间。”  
更强：“如果我重读同一段证据两次仍未得出新结论，则标记该题并继续。”

规则应能在一行内写完，并在下一次练习中可用。

## 区分知识缺口与流程缺口

知识缺口意味着你缺乏某个单词、语法规则或考试格式事实。流程缺口意味着你拥有足够的信息，但使用不当。

区别对待它们：

- 知识缺口：学习该项目，创建一个示例，然后在之后进行检索；
- 流程缺口：在遵循可见检查清单的同时重复类似任务；
- 时间缺口：在更短的限制下练习停止规则；
- 注意力缺口：识别错过的信号，并在下一组中突出显示该信号。

购买更大的词汇列表无法纠正忽视对比的习惯。重复整套测试不如专注于句子练习来高效解决特定文章错误。

## 安排检索

延迟复习错误，而不是立即重读解释。一个简单的计划是一天后、一周后和三周后。

每次复习时：

1. 隐藏旧答案；
2. 解答原题或平行题目；
3. 解释控制性证据；
4. 标记该规则为已掌握、薄弱或仍在失败。

如果同一规则失败两次，使修复动作更小。“绘制段落地图”可能需要变为“在每个段落旁边写三个字的职责说明”。

## 将每周数据转化为计划

在周末，统计错误类型而非总错误数。选择：

- 一个高频流程错误；
- 一个重要的知识缺口；
- 一个限时恢复技能。

围绕这三项构建下一周的计划。其他技能保持轻度维护。这可以防止单次糟糕的练习分数导致完全且不必要的计划变更。

## 示例条目

**任务：** IELTS 真/假/未提及 (True/False/Not Given)  
**我的决策：** 假 (False)  
**证据：** 文章提到了增加，但未给出原因。  
**错误类型：** 将缺失的证据视为矛盾。  
**修复规则：** 仅当我能从文本中陈述相反主张时，才选择假 (False)。  
**复习日期：** 明天，使用三个新的原因陈述。

此条目有用，因为它命名了一个可迁移的决策边界。

## 保持错题本诚实且精简

归档那些你在不同材料中正确应用过三次的规则。让反复出现的规则保持在页面顶部。活跃错题本应适合一屏或一页纸，以便它成为你真正使用的工具。

目标不是消除每一个错误答案。而是避免在未察觉的情况下重复相同的决策错误。
