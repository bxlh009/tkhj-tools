雅思G类阅读考试允许60分钟完成三个部分，且包含在该小时内进行答案誊写的时间。由于各部分的难度递增，将时间平均分为每部分20分钟是一个有用的起点，而非不可打破的规则。

## 在确定时间前建立基准

在严格限时下完成一套官方样题。记录每个部分的完成时间和准确率。你的计划应基于这些证据进行调整。

在纸上使用一个简单的表格：

| 部分 | 用时（分钟） | 正确数 | 未答题数 | 主要耗时原因 |
| --- | ---: | ---: | ---: | --- |
| 1 |  |  |  |  |
| 2 |  |  |  |  |
| 3 |  |  |  |  |

如果第一部分已经花费了25分钟，那么承诺给第三部分更多时间并不能解决问题。你需要一个“停止规则”。

## 从灵活的时间分配开始

一个实用的初步实验方案是：

1. 第一部分：15分钟；
2. 第二部分：18分钟；
3. 第三部分：22分钟；
4. 最后的答案誊写和拼写检查：5分钟。

这不是官方推荐的分配比例。这是一个可测试的计划，为篇幅更长、更复杂的最后一篇文章预留了更多时间，并保护了答案誊写的时间。

在复习两次完整的练习后，以两分钟为单位进行调整。大幅度的改变会让你难以判断哪些方法有效。

## 使用两轮做题法

在第一轮中，回答那些你能定位并验证证据的题目。对于卡住的题目做标记，如果在重读同一段落后仍未获得新信息，则继续前进。

在第二轮中，带着具体问题返回：我是否遗漏了同义替换、限定词或相关位置？重复相同的搜索过程不算作新的尝试。

对于纸笔考试，圈出题目编号。对于机考，使用官方界面中提供的回顾功能。你的方法应与你将参加的考试形式相匹配。

## 根据题型匹配阅读行为

不要将“先看问题”作为通用规则。它在定位人名、日期、条件和简短事实细节方面效果很好。但对于匹配标题或识别作者意图的题目，快速绘制段落大意可能更高效。

根据问题类型选择阅读动作：

- 细节题或简答题：扫描独特的锚点词，然后局部精读；
- 是非题（True/False/Not Given）：定位观点并精确比较含义；
- 匹配标题：识别每段的主要功能；
- 填空题：在搜索前预测所需的语法结构。

最快的方法是能收集到正确证据的方法，而不是眼球移动最快的方法。

## 制定补救规则

当计划没有应对困难的措施时，它就会失效。在练习前写下三条规则：

1. 如果一道题超过了我的时间限制，标记它并跳过。
2. 如果整篇文章感觉很难，先回答最容易看到的题目。
3. 如果剩余五分钟，优先完成答案誊写，然后再回头处理不确定的题目。

这些规则降低了单个难题消耗原本属于整个部分时间的可能性。

## 在最后阶段保护准确率

官方格式警告称，拼写和语法错误会影响得分，且不提供额外的阅读部分答案誊写时间。因此，最后的检查应专注于避免机械性失分：

- 每道题都有答案；
- 遵守字数限制；
- 单复数形式与原文一致；
- 人名和数字抄写准确；
- 答案填入正确的方框内。

不要在没有依据的情况下，用最后一分钟去重新思考一大组答案。

## 复盘计划，而不仅仅是分数

练习结束后，问自己遵循了哪条规则，以及计划在哪里失效。如果准确率高但有未答题目，需加强停止规则的训练。如果所有题目都完成了但准确率骤降，需放慢证据核查的速度。如果某种题型消耗了大量时间，在下一次全真模拟前，对其进行短时专项训练。

一个有用的时间计划在跨部分时具有灵活性，但在补救措施上则非常严格。它将注意力分配到证据显示你需要的地方。
